## Packages
Must be installed first:
```
npm i bcrypt
npm i cookie-parser
npm i express
npm i express-handlebars
npm i express-session
npm i mongodb
npm i seed
```
## Seed File
To populate database:
```
npm run seed
```
## Run File
To start the application run this command:
```
npm start
```
## Server
Routes will be running on:
```
http://localhost:3000
```
